  • Bachelor’s Final Year Project : ’An Elliptic Curve Cryptograpghy- Based Multi-server Authentication Scheme using Cancelable Biometrics.’
    We Designed an efficient and secured Authentication Scheme in multi-server Environment using biometrics.
    Used Elliptic Curve Cryptography (ECC) algorithm to provide security in this authentication scheme.
    Compared with some existing authentication schemes in terms of Security features and Computational times
    Used ProVerif tool to check the robustness of this scheme against different security attacks.
    Publised in Intelligent Computing and Communication (pp. 153-163) vol 1034. Springer, Singapore. https://doi.org/10.1007/978-981-15-1084-7
